========================================
 さくら整骨院 LP — 画像ファイル一覧
========================================

このフォルダに以下のファイル名で写真を入れてください。
拡張子は .jpg / .png / .webp どれでも構いません。
ファイル名は以下の通りに合わせてください。

────────────────────────────────────────

【1】 hero.jpg
  使用箇所 : ファーストビュー（ヒーローセクション）背景
  推奨サイズ: 1800 × 1000 px 以上
  比率      : 横長 16:9
  内容      : 下記のいずれか
    - 院内の清潔感ある待合室・施術室の風景
    - 白衣のスタッフが患者さんに丁寧に対応している施術風景
    - 明るく清潔感のある受付カウンター
  ポイント  :
    - テキストをメイン画像に重ねるため、中央〜左半分にスペースがある構図がベスト
    - 暗め・深めの色味の写真でも重なるオーバーレイで馴染みます
    - 「安心・清潔・信頼」が伝わる雰囲気を意識してください

────────────────────────────────────────

【2】 director.jpg
  使用箇所 : 院長挨拶セクション
  推奨サイズ: 560 × 480 px 以上
  比率      : 横長 7:6（ほぼ正方形）
  内容      : 院長 山本健太先生のポートレート写真
  ポイント  :
    - 白衣を着用した正面〜斜め45°のバストアップ
    - 笑顔・清潔感・誠実さが伝わる表情
    - 背景はサロン内・白背景・薄いグリーン系が映えます

────────────────────────────────────────

【3】 map.jpg   ※ Google マップ埋め込みにする場合は不要
  使用箇所 : アクセスセクションの地図エリア
  推奨サイズ: 700 × 500 px 以上
  比率      : 横長 7:5
  内容      : 下記のいずれか
    ① Google マップのスクリーンショット（院周辺）
    ② 手書き風・イラスト風のアクセスマップ
    ③ 外観写真（建物の正面）
  ポイント  :
    - Google マップの埋め込みコード（iframe）に差し替えると
      より使いやすくなります。ご希望であればコードを作成します。

========================================
 対応フォーマット
========================================
  .jpg  / .jpeg  → 推奨（軽量・Webに最適）
  .png           → 透過が必要な場合
  .webp          → 最も軽量（モダンブラウザ対応）

  ※ ファイルが重すぎるとページ表示が遅くなります。
     hero.jpg は 500KB〜1MB 以内に圧縮すると快適です。
     圧縮ツール例: https://squoosh.app/（無料・ブラウザで使用可）
